# Monitor de Máquinas Industriais - Faetec Mecânica RJ

**Projeto desenvolvido no 3º ano Técnico Mecânica Industrial Faetec RJ**

## Funcionalidades
- Monitoramento tempo real: temperatura, vibração, pressão
- Alertas automáticos: superaquecimento (>70°C), vibração alta (>5mm/s)
- Logs para manutenção preditiva
- Dashboard industrial IoT

## Aplicações
- Plataformas offshore Petrobras pré-sal
- Fábricas automação Embraer/Cosan

## Tecnologias
- Python 3
- Pandas (análise dados)
- Matplotlib (gráficos)

## Próximos Passos
- AWS IoT Cloud
- Banco PostgreSQL
- Sensores reais Arduino
